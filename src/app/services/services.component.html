Website Components

To have your website up and running you will need three components: domain, hosting, and code. I will help you with each of them.

Domain

Name for your website which is also called an URL. For example www.example.com

Average Price: $12 per year

Depends on the seller and extension (.com, .org, etc).

Hosting

Hosting provider is a fancy name for an online place where your website is stored and processed.

Average price: $5 per month

Depends on the hosting provider and duration of the agreement.

Code

Website itself which is files with the code.

Price: $75-360

Depends on the complexity.
